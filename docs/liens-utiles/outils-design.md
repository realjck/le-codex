# Outils pour le design

### [figma.com](https://www.figma.com/fr/)
Principal outil de design collaboratif pour la création

### [nerdfonts.com](https://www.nerdfonts.com/)
Polices destinées aux développeurs avec un nombre élevé de glyphes

### [pigment.shapefactory.co](https://pigment.shapefactory.co/)
Une façon unique de générer des couleurs basée sur l'éclairage et les pigments

### [realfavicongenerator.net](https://realfavicongenerator.net/)
Générateur de favicon pour votre site web.

### [shields.io](https://shields.io/)
Service de badges qui peuvent facilement être inclus dans les readme GitHub ou toute autre page Web

### [supercolorpalette.com](https://supercolorpalette.com/)
Générateur de palettes de couleurs. Modifiez la teinte, la saturation, la luminosité et bien plus encore !
