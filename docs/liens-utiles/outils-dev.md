# Outils pour le développement

### [app.diagrams.net](https://app.diagrams.net/)  
Réalisez toutes sortes de diagrammes pour vos applications

### [dbdiagram.io](https://dbdiagram.io/home)  
Outil pour vous aider à dessiner vos diagrammes de relations de base de données

### [gitmoji.dev](https://gitmoji.dev/) 
Un guide d'emojis pour vos messages de validation Git

### [hoppscotch.io](https://hoppscotch.io/)  
Testez vos API en ligne (alternative à Postman)

### [readme.so](https://readme.so/editor)  
Créez un fichier README rapidement avec cet éditeur en ligne
